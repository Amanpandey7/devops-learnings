What is a process?
A process is anything which performs a task successfully or unsuccessfully. 
It can be given an input or it gives an output, both optional.
Difference between absolute and relative path?
An absolute path is path of a file/directory from home/root directory while relative path is path from current directory.
What is shell script used for?
A shell script is used for performing bash operation automatically/programatically.
